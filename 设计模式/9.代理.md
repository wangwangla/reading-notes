# 代理模式

## 我和代理

```java
interface Work{
    public void work();
}
```

```java
我

class My implements Work{
    public void work(){
        sout("work……");
    }
}
```

```java
代理
class Proxy implements Work{
    private My my;
    public proxy(My my){
        this.my = my;
    }
    public void work(){
        my.work();
    }
}
```
