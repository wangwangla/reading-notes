## 可行性背包。

开始0个物体，0重量为true  0物体不0重量为false.

- 一个物品可以存放多满，n个物品可以存放多满。
- 如果重量小于物体，如果大于物体。

```java
//n个物品   w重量
dp[][] = new int[n+1][w+1];
for(i -> w){
    dp[0][i] = false;
}
dp[0][0] = true;
for(int i = 1;i<= n;i++){
    for(int j =0;j<=w;j++){
        //前i-1个已经得到了j值
        dp[i][j] = dp[i-1][j];
        if(j>=arr[i])
            //使用i-1个  或者i-1个加上第i个，
        	dp[i][j] = dp[i][j] | dp[i-1][j-arr[i]];
    }
}
```

## n个数据，多少种方式可以组成目标

n个数据，目标

```java
int dp[][] = new int[n+1][m+1];
dp[0][0] = 1;
for(int i=0;i<=m;i++){
    dp[0][i] = 0;
}
for(int i=1;i<=n;i++){
    for(int j=0;j<=m;j++){
        dp[i][j] = dp[i-1][j];
        if(j>=arr[i][j]){
            dp[i][j] += dp[i][j-arr[i]];
        }
    }
}
```
