希尔排序，每次不是与前一个比较，而是与前三个比较。



原理：将数据进行分组，对分组之后的数据进行排序。

选定一个增长量h，按照增长量h作为数据分组的依据，对数据进行分组。

减少增长量，最小减为1，重复即可 。

<img src="images/image-20200309070519330.png" alt="image-20200309070519330" style="zoom:50%;" />



```java
int h=1;
while(h<数组的长度/2){
    h=2h+1;
}

减小规则 h=h/2;
```
